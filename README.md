
# Neural Networks: Zero to Hero — Implementations

Personal implementations from [Andrej Karpathy's Neural Networks: Zero to Hero](https://www.youtube.com/playlist?list=PLAqhIrjkxbuWI23v9cThsA9GvCAUhRvKZ) playlist.

## Covered so far

- **Bigram model** — character-level language model using bigram statistics and a neural approach
- **Multilayer Perceptron (MLP)** — building a deeper character-level model (Bengio et al. 2003)
- **Activations, gradients & batch normalization** — internals of training stability, dead neurons, initialization

## In progress

- Backprop Ninja (manual backpropagation)
- WaveNet-style deep MLP

## Notes

This repo is a work in progress and will be updated as I continue through the series.
